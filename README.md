# DSW-VidaFit

## essa atividade conta com duas telas ```Home``` e ```Calculator```, onde o usuário irá informar valores como: ```Peso``` e ```Altura```, e através dos valores informados ele retornará o IMC ( Índice de Massa Corporal ) do usuário.
